# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## プロジェクト概要

0〜5歳児向け 右脳活性化・記憶力トレーニング Webアプリ。  
家庭で手軽に使える**完全無料・完全静的・バックエンドなし**のシングルページアプリ。  
七田式右脳教育の考え方に着想を得た個人運用ツール（「七田式」「しちだ」名称は使用しない）。

## 技術スタック

| 項目 | 内容 |
|------|------|
| フロントエンド | HTML / CSS / JavaScript（フレームワークなし・依存最小） |
| 音声読み上げ | ブラウザ標準 Web Speech API（`speechSynthesis`）— APIキー不要・完全無料 |
| データ保持 | `localStorage`（設定値のみ）— DB・サーバー不要 |
| オフライン | PWA（Service Worker）でアセットをキャッシュ |
| ホスティング | GitHub Pages / Netlify / Cloudflare Pages（無料静的ホスティング） |
| 印刷 | `window.print()` + `@media print` CSS |
| バックエンド | **なし**（認証・DB・課金すべて不要） |

外部APIは使用しない（Supabase・Stripe・NextAuth等はこのプロジェクトでは不使用）。

## サイト構成

**1ページ縦スクロール型（ランディングページ形式）**。上から順に：

1. ヒーロー＋右脳トレーニングの説明（臨界期・高速大量入力の意義）
2. フラッシュカード セクション（意義・やり方・[はじめる][印刷]）
3. ドッツ セクション（意義・やり方・[はじめる][印刷]）
4. イメージ・暗唱 セクション（意義・やり方・[はじめる][印刷]）
5. 右脳記憶ゲーム セクション（優先度低・後フェーズ）
6. フッター（免責・年齢別目安・出典）

「はじめる」を押すと全画面トレーニングモードへ。「←」で1ページ構成に戻る。

## アニメーション要件

- **スクロール連動の出現:** Intersection Observer でセクションがふわっとフェードイン＋せり上がり
- **フラッシュカード切替:** 軽いポップ（拡大＋わずかな回転）でめくり感
- **ドッツ:** 点が1つずつ時間差でポンと現れる
- **ホバー/タップ:** ボタン・タイルに弾むようなスケール反応
- **原則CSSアニメーション**（重いライブラリ不使用）
- `prefers-reduced-motion` を必ず尊重する

## カラー印刷ページ

各コンテンツに「印刷」ボタンを設け、ブラウザの印刷機能で紙カードとして出力できる。

- `@media print` で画面用装飾・ボタン・背景を非表示
- `print-color-adjust: exact` でカード背景色・イラストを正確に印刷
- A4カラー印刷基準。裁ちトンボ／カット線（点線）を入れる
- 1ページに複数配置（例：2×2=4枚、2×3=6枚）

## 重要な制約

- 「七田式」「しちだ」の名称をコード・公開ページに使用しない（「右脳トレーニング」等の一般名称を使う）
- 公式教材の画像・音声・語彙リストを転載しない
- 効果の断定表現を避ける（「〜とされる」「〜と考えられている」と表記）
- 個人情報を収集・送信しない
