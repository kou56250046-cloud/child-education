0〜5さいの きおく・ちょっかんトレーニング

右のう 感性・イメージ・直感
ひらめきの脳
左のう 言語・論理・計算
ことばの脳

右脳が活発な「今」がチャンス — 臨界期イメージ

右脳は感性・イメージ・直感、左脳は言語・論理・計算をつかさどるとされています。両方をバランスよく育てることで、高い力を発揮できると考えられています。

右脳は生まれてすぐは活発に働いており、成長とともに左脳が優位になっていくとされています。右脳が活発な乳幼児期に楽しい刺激をたっぷり与えることが大切と考えられています。

1枚1秒以下の高速提示で、瞬時に処理しようとする右脳が働き始めて活性化するとされています。

※ここで紹介している効果は、右脳教育に関する教育理論の紹介であり、医学的・科学的に確立された効果を保証するものではありません。お子さまの様子を見ながら、楽しく取り組むことを第一にしてください。

おすすめ

フラッシュカード

なぜやるの？

絵や文字のカードを高速・大量に見せることで、瞬時に処理する右脳が働き始めて活性化するとされる、右脳トレーニングの基礎です。語彙力・理解力・表現力が育つとされます。

こうするといいよ

1枚1秒以下のテンポでリズムよく見せましょう。嫌がったらすぐやめることが大切です。毎日少しずつ同じカードを繰り返すことで定着しやすくなります。

おすすめ

ドッツ

なぜやるの？

赤い点の数を高速で見せることで、右脳の計算力・数量感覚を目覚めさせるトレーニングです。数を「量」として直感的にとらえる力を育てるとされます。

こうするといいよ

点の配置を毎回ランダムにし、高速でテンポよく見せながら数字を声に出して添えましょう。少ない数から始め、段階的に増やしていきます。

🌈
中おすすめ

イメージ・あんしょう

なぜやるの？

短い詩や言葉を繰り返し唱えて倍速で言えるようにすることで、記憶の質を高め、完全記憶を育てるとされます。ありありと想像するイメージ力も育てます。

こうするといいよ

短い詩・童謡から始め、毎日繰り返しましょう。覚えたら2倍速・3倍速と速めて記憶の質を高めます。完璧を求めず、リズムや音の楽しさを大切に。

🧩
あたらしい

メモリーチップ

なぜやるの？

チップを使って記憶力・記憶法を楽しく育てるトレーニングです。順番に覚えて再現したり、神経衰弱形式でペアを探したり、仲間に分けたりして右脳の記憶力を育てます。

こうするといいよ

リンク法ではチップを順番に示しながら情景的なお話を作りましょう。神経衰弱は枚数を少なくからスタートし、慣れたら増やします。

みぎのう君

なぜやるの？

絵・色・記号のチップを使って「完全記憶」を育てます。グリッドに並べたチップを覚えて再現するマス記憶や、ポーカーゲームで楽しく記憶力を鍛えます。

こうするといいよ

マス記憶は最初は少ないチップ・短い時間から始め、できたら徐々に難しくしましょう。当てる楽しさ・できた喜びを大切に、遊び感覚で取り組みます。

みぎのう あそび について

0〜5歳児向けの右脳活性化・記憶力トレーニング Webアプリです。個人運用の学習補助ツールであり、特定の教育機関・企業とは関係ありません。

音声にはブラウザ標準の Web Speech API を使用しています（無料・APIキー不要）。

本アプリはオフラインでも使用できます（PWA対応）。スマートフォンの「ホーム画面に追加」でアプリのようにご利用いただけます。
